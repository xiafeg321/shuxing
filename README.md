# 数星 - 网页原型

## 🌟 项目简介
"再陪我数次星星"（简称"数星"）是一个情感陪伴网页应用，帮助经历断崖式分手的人群度过情感戒断期。

## 🚀 快速开始

### 方法一：使用Node.js本地服务器（推荐）
```bash
# 1. 确保已安装Node.js
node --version

# 2. 启动开发服务器
node server.js

# 3. 在浏览器中访问
# 本地：http://localhost:3000
# 网络：http://[你的IP地址]:3000
```

### 方法二：直接打开HTML文件
```bash
# 1. 使用浏览器直接打开
# 双击 index.html 或 setup.html
# 注意：某些功能可能需要本地服务器支持
```

### 方法三：使用Python简单服务器
```bash
# Python 3
python3 -m http.server 3000

# Python 2
python -m SimpleHTTPServer 3000
```

## 🎯 核心功能

### 已完成功能
1. **首页** (`index.html`)
   - 温暖治愈系UI设计
   - 响应式布局（支持手机/电脑）
   - 动态星光背景
   - 功能介绍和使用说明

2. **人格设置页面** (`setup.html`)
   - 星座选择（12星座）
   - MBTI类型选择（16种）
   - 聊天记录输入
   - 进度指示器
   - 本地存储（保存用户选择）

### 待开发功能
1. **对话页面** (`chat.html`)
   - 两种对话模式：陪伴/咨询
   - 聊天界面
   - 规则引擎回复
   - 对话历史记录

2. **人格分析页面** (`analysis.html`)
   - 详细人格分析
   - 情感建议
   - 成长路径

## 🎨 设计特点

### UI/UX设计
- **温暖治愈系**：浅蓝/浅紫配色，圆角设计
- **响应式布局**：完美适配手机和电脑
- **交互动画**：平滑过渡，微交互反馈
- **无障碍设计**：支持键盘导航，屏幕阅读器

### 技术特点
- **纯前端实现**：HTML5 + CSS3 + JavaScript
- **本地存储**：使用localStorage保存用户数据
- **模块化代码**：清晰的代码结构
- **渐进增强**：基础功能可用，高级功能增强

## 🔐 隐私与安全

### 数据存储
- **本地存储**：所有数据存储在用户浏览器中
- **无后端**：不发送数据到服务器
- **可清除**：用户可随时清除浏览器数据

### 隐私保护
- 不收集个人信息
- 不跟踪用户行为
- 无第三方分析工具
